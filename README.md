## Run Locally

### In Terminal
1. `cd client`
2. `npm run build`
3. `cd ..`
4. `npm start`
5.  open browser and search http://localhost:5000/


#### If Error Occurs While Running, try
1. `npm install`
2. `cd client`
3. `npm install`
4. `cd ..`
5. `npm run dev`
